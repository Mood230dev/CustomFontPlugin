# рЯУЭ CustomFontPlugin

CustomFontPlugin is a lightweight and optimized Spigot/Paper plugin that automatically converts player chat messages into a custom font (small caps style). 

It is specially designed to add a unique and immersive atmosphere to the chat interface in survival, RPG, and custom concept servers.

## вЬ® Features
* **Automatic Font Translation:** Instantly translates normal messages written by players into special characters like `«Ђбі°біЗ АбіЫ ПбіЬ…™біПбіШбіА—ХбіЕкЬ∞…Ґ ЬбіКбіЛ ЯбіҐ—ЕбіДбі† Щ…ібіН`.
* **Full Color Code Support:** Fully supports classic Minecraft color codes (like `&a`, `&c`, `&l`, etc.). The color codes remain intact while the rest of the text is converted to the custom font.
* **Discord Webhook Integration:** In-game chat messages can be seamlessly forwarded to your specified Discord channel in real-time.
* **Lag-Free Performance:** Runs asynchronously using `AsyncPlayerChatEvent`, ensuring zero impact on the main server thread.
